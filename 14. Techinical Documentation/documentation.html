Intro to Python

What is Python?

Python is a popular programming language. It was created by Guido van Rossum, and released in 1991.

It is flexible and some usages include:

  1. web development (server-side)
  2. software development
  3. mathematics
  4. system scripting

What can Python do?

To cite a few of an evergrowing list of usages, Python can be used on a server to create web applications and alongside software to create workflows. It can also connect to database systems, read and modify files, and handle big data and perform complex mathematics. Lastly, Python can be used for rapid prototyping, or for production-ready software development. As described, one can see how Python is flexible to encompass a wide array of usages in modern systems.

Why Python?

Python works on different platforms (Windows, Mac, Linux, Raspberry Pi, etc) and has a simple syntax similar to the English language. It has syntax that allows developers to write programs with fewer lines than some other programming languages. It runs on an interpreter system, meaning that code can be executed as soon as it is written, making it a great language for prototyping. Moreover, Python can be treated in a procedural way, an object-oriented way or a functional way.

Good to know

The most recent major version of Python is Python 3, which we shall be using in this sample documentation page. However, Python 2, although not being updated with anything other than security updates, is still quite popular. It is possible to write Python in an Integrated Development Environment, such as Thonny, Pycharm, Netbeans or Eclipse which are particularly useful when managing larger collections of Python files.

Python Syntax

  • Python was designed for readability, and has some similarities to the English language with influence from mathematics.
  • Python uses new lines to complete a command, as opposed to other programming languages which often use semicolons or parentheses.
  • Python relies on indentation, using whitespace, to define scope; such as the scope of loops, functions and classes. Other programming languages often use curly-brackets for this purpose.

Python's intuitive grammar can be seen, for example, in a line of code that prints some text print("Hello, World!")

Given that Python is a language with many features, writing an extensive tutorial would be a very time consumin endeavor, filled with research, trial and error, library explorations and such. With that in mind, this intro to Python aims to expose the reader to some important data types available. Happy Coding!

Built in Data Types

In programming, data type is an important concept.

Variables can store data of different types, and different types can do different things.

Python has the following data types built-in by default, in these categories:

Text Type: str
Numeric Types: int, float, complex
Sequence Types: list, tuple, range
Mapping Type: dict
Set Types: set, frozenset
Boolean Type: bool
Binary Types: bytes, bytearray, memoryview
None Type: NoneType
